# Inventory-first homepage

## Goal

Reorder and simplify the public homepage so visitors reach property search and
featured listings sooner, while retaining the existing buy, rent, sell, and
direct-contact journeys.

## Scope

- Simplify the public header into one primary navigation row.
- Refocus the hero on available property types and location.
- Place search immediately after the hero and listings directly after search.
- Reduce buy/rent/sell into a compact routing section.
- Replace the generic services ending with a more personal service statement
  and direct WhatsApp action.
- Preserve the existing neutral palette, property data, and user-authored copy
  adjustments already present in the working tree.

## Verification

- `npm run lint`
- `npm run build`
- Desktop and mobile visual review in the in-app browser
- Basic navigation and property-search interaction check
